# Test Data

This directory contains test data for AMD-BioForge.

## Files

- `1ubq_test.pdb` -- Small test protein (ubiquitin fragment)
- `sample_ligands.sdf` -- Sample ligand library (coming soon)

## Download Full Data

```bash
bash scripts/download_data.sh
```

## Data Sources

- PDB structures: [RCSB PDB](https://www.rcsb.org/)
- Ligand libraries: [ZINC](https://zinc.docking.org/)
- Binding data: [PDBbind](http://www.pdbbind.org.cn/)
